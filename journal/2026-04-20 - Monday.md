### What I learned
- To delete folder, i learnt that i had to use rm -rf folderName. However this command is permanent with no retrieval and deletes all folders and file in the folderName. I use it to delete a zoom recording folder which took some time but eventually worked.
- less shows a text file line by line
- head shows the first 10 lines of a text file. However, we can do `head -n 5 fileName` to print the first 5 lines of the file.
- tail shows the last 10 lines of a text file
- I learnt that using tail -f textFile shows the text file in live view. also it is important for security work as we can see attack attempts in real time if we use it on /var/log/auth.log
- While writing file, using > overwrites the entire text file while using >> appends the text to be written to the next line hence, we should be careful.
- Executable commands like ls ,grep, head, tail are called binaries and are found in the usr/bin and usr/sbin
- Linux is case sensitive
- Root is a administrator/superuser account that is used by a trusted person and has absolute control of the entire system. The command `whoami` would reutrn root if you are the root user
- scripting tools are python .ruby, perl
- The most widely used shell is the Bash shell while other popular shells are the C shell and Z shell

- Linus uses a logical filesystem. The root / is different from the root user
- /etc contains Linux configuration files that controls when and how programs start up
- `cd` means change directory
- To move multiple directory towards the root, you can use more than one double .. e.g `cd .. ..` moves two levels upwards towards root
- `ls -l` stands for long listing and is used to get more information about file and directories (folders) such as permissions, number of links, owner, group, size, and creation or last modification date
