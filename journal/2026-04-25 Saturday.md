### What I tried
- Moving cursor in Vim using keys h (left), j (down), k (up) and l (right). The arrow keys also work.
- Using the command `:q!`, vim exits without saving
- While the cursor hovers over a letter, we can press `x` to delete that letter
- To insert into the text, press `i` to go into insert mode
- Using `A` appends text to the end of the current line
- Using the command `:wq` saves and exits from the text file
- Pressing `esc` will always move to normal mode
- Moving cursor to the first letter of a word, press `dw` to delete a word till the start of the next word, excluding its first character
- Moving cursor to the first letter of a word, press `de` to delete to the end of the current word, including the last character
- From anywhere in the text, press `d$` to delete till the end of the line
- Pressing `2w` moves the cursor two words forward, `3e` moves the cursor 3 words forward to the end of the third word while `0` moves the cursor to the start of the line. Signs like `,.!` all count as a word.
- Adding `d` to `2w` or `3e` deletes words that it should normally traverse.
- We can delete a whole line by typing `dd`, two whole lines with `2dd` and so on
- We can use `u` to undo previous commands, `U` to fix a whole line and `ctrl + R` to redo the commands i.e undo the undos
